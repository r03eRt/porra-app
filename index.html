Proyecto independiente

Porra Admin Next

Base de trabajo para montar una porra nueva sin depender del Excel ni de la edición actual.

Acceso admin

Apunta este proyecto a un Supabase nuevo usando `public/admin-next-config.js`.

Estado

Borrador local persistido en este navegador. La siguiente fase es persistirlo entero en Supabase.

Configuración de la porra

Nombre, edición y reglas de puntuación base.

Participantes

Base de alta de jugadores y administradores de la porra.

Grupos y selecciones

Configura los grupos y los países que pertenecen a cada uno.

Mini-porra

Preguntas, tipo de respuesta y puntos asociados.

Cruces

Definición base de rondas y slots para la eliminatoria.
